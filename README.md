# ✨ Smart To-Do App

A beautiful, modern to-do list application with a glassmorphism design and smooth animations.

## Features

- ✅ **Add Tasks** - Quickly add new tasks with a clean input interface
- ✏️ **Edit Tasks** - Update task text with one click
- 🗑️ **Delete Tasks** - Remove completed or unwanted tasks
- ✓ **Mark Complete** - Check off tasks as you complete them
- 📊 **Statistics Dashboard** - Track total, completed, and pending tasks
- 🔍 **Smart Filtering** - Filter tasks by All, Pending, or Completed status
- 💾 **Local Storage** - All tasks are saved locally in your browser
- 📱 **Responsive Design** - Works perfectly on desktop and mobile devices
- 🎨 **Beautiful UI** - Modern glassmorphism design with gradient backgrounds
- ✨ **Smooth Animations** - Fade-in, slide-up, and hover effects

## How to Use

1. Open `index.html` in your web browser
2. Enter a task in the input field and click "Add Task" or press Enter
3. Use the filter buttons to view All, Pending, or Completed tasks
4. Click the ✏️ button to edit a task
5. Click the 🗑️ button to delete a task
6. Check the checkbox to mark a task as complete/incomplete
7. View your task statistics at the top of the app

## Technologies Used

- **HTML5** - Markup structure
- **CSS3** - Styling with animations and gradients
- **Vanilla JavaScript** - Task management and DOM manipulation
- **localStorage API** - Data persistence
